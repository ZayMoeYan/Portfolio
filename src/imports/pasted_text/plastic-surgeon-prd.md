Build a **high-end, modern, and professional personal branding website** for a Plastic Surgeon based on the following PRD.

Project: **Plastic Surgeon Personal Branding Website**
Website Type: **Portfolio / Personal Branding Website**

---

# 1. Project Goal

Develop an **international-standard, premium-quality website** that:

• Establishes strong **trust and authority**
• Attracts **international patients and partners**
• Elevates the doctor’s **personal brand**
• Drives **consultation inquiries and conversions**

Target users:

• International patients
• High-end local clients
• Medical partners
• Media / PR teams

---

# 2. Design & Branding

### Design Style

• Clean, minimal, and highly professional
• Premium medical aesthetic
• Spacious layout with strong visual hierarchy
• Soft shadows and smooth rounded corners
• Subtle animations (not distracting)

---

### Color System

Use the following palette consistently:

• #0d1b2a → Primary background
• #1b263b → Secondary background
• #415a77 → Cards / sections
• #778da9 → Secondary text
• #e0e1dd → Main text

---

### Typography

• Primary: Roboto
• Secondary: Roboto Mono
• Fallback: Monospace

---
